# TG电报群搜索机器人推荐：如何搜索想要的群组和频道

众所周知，[Telegram](https://telegram.org)的消息搜索功能在处理东亚语言或任何不使用空格来分隔单词的语言时，无法正常工作。在英语中，单词之间通过空格进行分隔；但在中文、日文等语言中，句子内的单词之间既不使用空格也不使用其他分隔符。因此可以得出结论，Telegram 目前仅能识别并搜索那些被特定分隔的单词，这对中文和日文用户而言极其不便，甚至到了几乎无法使用的地步。而上述情况已经在Telegram 的 macOS、Android 以及网页版本上都得到了验证。

还好在这样情况下，电报群搜索机器人应运而生，帮你自动搜索并展示结果。而使用TG群组搜索机器人非常简单，只需在搜索栏输入机器人用户名（如 [@TGdotCoolBot](https://t.me/TGdotCoolBot)），点击“START BOT”，然后在对话框中输入并发送你想要查找的关键词，即可批量获取相关的频道和群组链接。
